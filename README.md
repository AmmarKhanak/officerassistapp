# Officer_Assistant

Small Python project with `app.py`, `db_manager.py`, and `email_handler.py`.

How to use

1. Create a virtual environment: `python -m venv .venv`
2. Activate it (PowerShell): `.\.venv\Scripts\Activate.ps1`
3. Install dependencies (if any): `pip install -r requirements.txt`
4. Run `app.py`: `python app.py`

Repository setup: see repository instructions to push this local project to GitHub.
